# 快看相册（Windows）

轻量现代的看图工具：极速缩略图、原图独立窗口预览、滚轮缩放、键盘切换。

## 快速开始
1) 安装依赖（Python 3.9+）  
   `pip install -r requirements.txt`
2) 在你的图片目录下运行  
   `python main.py`  
   或先运行后点击「选择文件夹」切换目录。

## 使用
- 缩略图大小：左上角下拉（小 / 中 / 大 / 特大）
- 打开原图：单击/双击缩略图，或点击“打开原图”按钮（也可 Enter）
- 缩放/复位：鼠标滚轮缩放，空格或 Esc 适配窗口
- 切换：键盘 ←/→ 或 A/D；原图窗口顶部按钮也可切换
- 拖拽：按住左键拖动画布

## 性能
- 缩略图多线程生成 + LRU 缓存，优先秒开列表
- 原图加载独立线程 + 缓存，切换和放大更流畅
- 高 DPI 适配，默认保持原比例、适配窗口
